= Welcome to Spi

== Getting Started

* Install Ruby at the command prompt if you haven't yet:
  
    $ rvm install 2.2.1

* Install Ruby Gems
  
    $ bundle install

* Create Database
  
    $ rake db:create

* Run Migration
  
    $ rake db:migrate

* Create Admin User
    
    $ rake db:seed

* Build the Sample Data
    
    $ rake sample_data:build_all
